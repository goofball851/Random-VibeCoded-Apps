PocketWeb is a professional-grade web application wrapper designed to consolidate multiple web tools into a single, highly customizable interface. It leverages a modern React stack with an emphasis on aesthetics, performance, and user-centric configuration.

- **Multi-App Management**: Run multiple web apps (Gmail, Notion, Slack, GitHub, AI tools, etc.) in a unified workspace with side-by-side management.
- **Persistent State**: Automatically saves your workspace layout, active apps, and theme preferences to `localStorage`.
- **Advanced Customization Engine**:
  - **Luminance Engine**: Sliding toggle for Dark and Light modes.
  - **Interface Density**: Choose between Comfortable and Compact UI layouts.
  - **Theming**: Select from curated presets (Deep Space, Nordic Frost, Cyberpunk, etc.) or create a custom palette.

  - **Landing Synthesis**: Fully customizable home area with typography controls (Font family, weight, style, scale) and positioning.
- **Productivity Tools**:
  - Global zoom level control for hosted apps.
  - Instant refresh and external link access.
  - Quick-search tool library for easy instance injection.
- **Mobile Responsive**: Adaptive design that works seamlessly across desktops, tablets, and mobile devices with a dedicated mobile navigation system.

## ðŸ“¦ Installation & Usage

Since this is an ESM-based web application, you can simply open the `index.html` in a modern browser.

1. Clone the repository.
2. Ensure `PocketWeb.png` is present in the root directory for the default experience.
3. Open `index.html` via a local server (like Live Server or `python -m http.server`).
4. Click **"Initialize System"** to start adding your favorite tools.
5. Access **Preferences** (bottom-left gear icon) to synthesize your ideal atmosphere and interface.

### Adding New Tools
You can add tools from the **Preset Cluster** or inject any URL via the **Direct Instance Inject** form in the "Assemble Tool" modal.

### Theme Synthesis
Navigate to the **Theme** tab in Preferences to:
- Override accent and text colors.
- Toggle between Light and Dark luminance profiles.
- Switch between various design paradigms (Cyberpunk, Minimalist, Royal Crimson).

### Environmental Atmosphere
Navigate to the **Atmosphere** tab to:
- Upload your own background textures.
- Adjust **Matrix Blur** for a focused workspace.
- Control **Alpha Channels** for background visibility.
