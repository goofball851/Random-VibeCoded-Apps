## How to Use

### Creating Your Video

1. **Add Photos**
   - Click "Add Photo Frame" to insert a new frame
   - Click "Upload Image" to add your own photos
   - Or use placeholder images for testing

2. **Customize Frames**
   - Click on any frame in the timeline to select it
   - Adjust duration using the slider (0.5s - 10s)
   - Change background color with the color picker
   - Choose a transition effect from the dropdown

3. **Add Text**
   - Type text in the "Overlay Text" field
   - Adjust text size and color
   - Choose text position (top, center, bottom)
   - Apply text effects for extra style

4. **Apply Effects**
   - Navigate to the "Effects" tab
   - Select photo filters (grayscale, sepia, etc.)
   - Choose text effects (shadow, neon 3D, etc.)
   - Add motion effects (glitch, color shift, flipbook)

5. **Add Countdown** (Optional)
   - Click "Add Countdown" button
   - Set starting number and style
   - Customize duration and background

6. **Preview Your Video**
   - Click the play button to preview
   - Watch your video with all transitions and effects
   - Make adjustments as needed

7. **Export** (Coming Soon)
   - Export functionality to be implemented
   - Will support multiple video formats

## Technical Details

- **Next.js 16** - React framework with App Router
- **React 19** - Latest React features
- **TypeScript** - Type-safe development
- **Tailwind CSS v4** - Modern utility-first styling
- **Framer Motion** - Smooth animations
- **shadcn/ui** - Beautiful component library


### Key Concepts

**Frame Types:**
- Photo frames with images, text, and effects
- Countdown frames with animated numbers

**Timeline System:**
- Sequential frame playback
- Customizable duration per frame
- Smooth transitions between frames

**Effect System:**
- CSS-based photo filters
- Text shadow and transform effects
- Keyframe animations for motion effects

**Transition Timing:**
- All transitions are 0.8 seconds
- Frames display for custom duration
- Smooth animation using Framer Motion
